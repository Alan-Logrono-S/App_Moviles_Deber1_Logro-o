# Calculadora de Notas EPN 🎓

Una aplicación web desarrollada como herramienta académica para calcular el estado de aprobación de los estudiantes de la Escuela Politécnica Nacional (EPN), determinando si aprueban directamente, reprueban o necesitan rendir un examen supletorio.

Este proyecto fue generado y refinado utilizando **OpenCode IA** como asistente de desarrollo.

🚀 Características

* **Ingreso de Notas:** Permite registrar las calificaciones del Primer y Segundo Bimestre (sobre 20 puntos cada uno).
* **Lógica de Aprobación EPN:**
  * **Aprobado:** Si la suma de los dos bimestres es mayor o igual a 28 puntos.
  * **Supletorio:** Si la suma es menor a 28 puntos, el sistema calcula automáticamente la nota exacta que el estudiante necesita obtener en el examen supletorio para aprobar la materia.
* **Validación de Datos:** Evita el ingreso de notas negativas o mayores a 20.

🛠️ Tecnologías y Herramientas Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
* **Inteligencia Artificial:** OpenCode (Generación de estructura base, refinamiento de lógica de negocio y estilos).
* **Control de Versiones:** Git y GitHub.
